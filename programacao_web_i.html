Programação Web I

A disciplina de Programação Web I tem como objetivo introduzir os conceitos fundamentais de desenvolvimento web, abordando linguagens, tecnologias e técnicas utilizadas na criação de páginas e aplicações web.

Objetivos

  • Compreender os princípios básicos da arquitetura web.
  • Aprender linguagens de marcação como HTML
  • Explorar folhas de estilo com CSS para estilizar páginas web.
  • Desenvolver habilidades em linguagens de programação do lado do cliente, como JavaScript.
  • Introduzir o conceito de desenvolvimento responsivo e acessível.

Conteúdos Abordados

  • Introdução à web e arquitetura cliente-servidor.
  • Linguagem de Marcação HTML5 e sintaxe básica.
  • Folhas de Estilo CSS3 e seletores.
  • JavaScript: manipulação do DOM, eventos e validação de formulários.
  • Desenvolvimento responsivo e adaptativo.
  • Acessibilidade na web.
